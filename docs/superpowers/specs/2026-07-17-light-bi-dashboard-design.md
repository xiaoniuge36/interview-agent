# 轻量 BI 数据看板设计

## 目标

将平台数据看板从指标卡片与手绘 SVG 的组合，升级为适合日常经营决策的轻量 BI 仪表盘。页面保持在现有后台 `#analytics`，继续展示真实平台数据，不新增独立系统或伪造统计值。

## 选型

引入 `@ant-design/charts@^2.6.5`。它是 AntV 的 React 图表库，与现有 Ant Design 和 React 18 技术栈兼容；图表组件在客户端动态加载，避免 Canvas/G2 在 Next 服务端渲染和静态测试环境中执行。

## 页面结构

1. **运营概览**：取消整块深蓝横幅。使用白色卡片容器，左侧为平台健康状态与说明，右侧为账号总量、窗口新增、训练完成、平均延迟四个紧凑指标。关键数字用深色文字与小面积品牌蓝强调。
2. **经营趋势**：使用 `Area` 图展示选定指标的按日趋势；提供 Agent 调用、训练完成、新增发布题、新增账号切换。图表支持 Tooltip、平滑渐变、真实日期轴和零值显示。
3. **运行质量**：使用 `Gauge` 展示 Agent 成功率，旁边保留 Schema 通过率、延迟、降级次数；进度条只作为辅助，不再占据主要视觉面积。
4. **内容与训练链路**：使用横向 `Bar` 图展示导入、待审核、已发布、训练提交、练习报告；下方显示发布率与报告完成率。待审核积压和失败导入转化为可读的风险提示。
5. **运行风险**：无风险时使用紧凑空状态；有风险时保留最近失败表格，并通过严重级别和操作建议突出需要处理的事项。

## 视觉原则

- 背景使用冷灰 `#f5f7fa`，卡片白色，分隔线 `#edf1f5`。
- 主数据色为治理蓝 `#1677ff`，辅助使用青色 `#13a8a8`、琥珀色 `#d48806` 和告警红 `#cf1322`，每个图表只突出一类主指标。
- 统一 12px 圆角和 16px 区块间距；降低渐变、阴影和大面积饱和色的使用。
- 图表仅在数据更新时执行短暂入场动画；支持键盘焦点、Tooltip 和无障碍说明。

## 约束与验证

- `PlatformDashboard` API 和 Prisma 聚合保持不变；只消费现有 `trend`、`funnel`、`alerts` 和 `runtime` 数据。
- 新增图表组件与模型的单元测试；静态渲染测试 mock 图表库，验证关键标题、数据和空状态。
- 验证桌面与移动端布局、周期切换、图表 Tooltip、控制台无错误、后台 TypeScript/Lint/全量测试通过。
