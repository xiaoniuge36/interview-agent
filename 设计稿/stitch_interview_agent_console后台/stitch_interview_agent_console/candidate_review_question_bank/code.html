GENERATED QUESTION

什么是 LangGraph StateGraph？请描述其核心组件以及它如何维护代理应用中的状态。

REFERENCE ANSWER (RAG GENERATED)

LangGraph 的 StateGraph 是一个用于定义循环计算图的核心类，专门用于构建具备状态的代理（Agents）。 它的核心是将一个 `State` 类型（通常是 TypedDict 或 Pydantic 模型）传递给图中的每个节点。

核心组件包括：
  • Nodes (节点): 代表计算步骤，接收当前状态并返回更新。
  • Edges (边): 定义计算流，包括条件边用于路由逻辑。
  • State (状态): 在节点间流动的数据，每次节点执行后进行更新（覆盖或追加，取决于 reducer）。
